# 📚 Book Keeper API

A simple RESTful API to manage a collection of books using Node.js, Express, and MongoDB.

---

## 🚀 Features

- Add a new book 📖
- View all books 🧾
- Update a book ✏️
- Delete a book ❌
- MongoDB integration 📦

---

## 🛠 Tech Stack

- Node.js
- Express.js
- MongoDB (local or Atlas)
- Mongoose
- Postman / cURL for testing
